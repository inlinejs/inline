## Component Usage

> The following example is using the `inline-grid` to contain a group of `inline-swatch` components.

<inline-grid grid-template-columns="1fr 1fr">
  <inline-swatch set="brand" color="primary"></inline-swatch>
  <inline-swatch set="status" color="success"></inline-swatch>
  <inline-swatch set="slate" color="200"></inline-swatch>
</inline-grid>
